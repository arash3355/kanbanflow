# 🚀 KanbanFlow

KanbanFlow adalah aplikasi manajemen tugas berbasis web (Full-Stack Web Application) yang membantu pengguna mengelola pekerjaan menggunakan metode Kanban. Pengguna dapat membuat, mengubah, menghapus, dan mengelompokkan tugas berdasarkan status sehingga aktivitas menjadi lebih terorganisir.

---

# 📖 Deskripsi

Aplikasi ini dikembangkan sebagai Project Akhir mata kuliah **Pemrograman Web II**.

KanbanFlow menerapkan konsep Full-Stack Development dengan menggunakan:

- Frontend : Next.js + React + Bootstrap
- Backend : Express.js
- Database : PostgreSQL
- API : REST API

# ⚙️ Cara Menjalankan Backend

Masuk ke folder backend

```bash
cd backend
```

Install dependency

```bash
npm install
```

Jalankan server

```bash
npm run dev
```

Backend berjalan pada

```
http://localhost:3001
```

---

# ⚙️ Cara Menjalankan Frontend

Masuk ke folder frontend

```bash
cd frontend
```

Install dependency

```bash
npm install
```

Jalankan aplikasi

```bash
npm run dev
```

Frontend berjalan pada

```
http://localhost:3000
```

---

# 🗄️ Database

Database yang digunakan adalah PostgreSQL.

Pastikan membuat file `.env` pada folder backend.

Contoh konfigurasi:

```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=kanbanflow
DB_USER=postgres
DB_PASSWORD=password

PORT=3001
```
